扫描探测

ARP扫描/PING扫描

ARP协议完成IP地址和MAC地址的转换，询问目的机的MAC地址

通过ping扫描或arp扫描可以知道那些地址上有活动主机

TCP扫描

利用tcp协议，扫描tcp的开放端口。发送SYN包，回复SYN ACK就说明端口开放
